Animationen

transition

Mit der Eigenschaft transition lässt sich einstellen, dass bestimmte Eigenschaften, wenn der entsprechende Wert verändert wird (zum Beispiel da er innerhalb des Pseudoselektors :hover überschrieben wird) nicht einfach „springen“, sondern ein sanfter Übergang zwischen den Werten dargestellt wird.

div	{
	width: 5rem;
	transition: all 1s;
}

div:hover	{
	width: 10rem;
}

Die Eigenschaft transition erhält mindestens zwei, maximal vier Werte. Der erst gibt an welche Eigenschaft einen Übergang erhalten soll. Soll jede Eigenschaft, für die ein Übergang möglich ist, einen Übergang erhalten, so wird all angegeben, ansonsten wird der Name der Eigenschaft angegeben. Als zweites folgt die Dauer des Übergangs. Als optionale dritte Eigenschaft kann die Kurve, in der der Übergang erfolgt angegeben werden; mögliche Werte sind ease (Standardwert), linear, ease-in, ease-out, ease-in-out. Die optionale vierte Eigenschaft gibt an, mit welcher Verzögerung der Übergang beginnen soll, hier wird wiederum eine Dauer angegeben.

div	{
	transition: all 1s ease-in-out 2s;
}

Achtung, Nicht alle Eigenschaften können einen Übergang haben. Hier ist eine Liste der „animatable properties“.

Manchmal sollen verschiedene Eigenschaften verschiedene Übergänge erhalten. Die verschieden Werte werden dann mit Kommas getrennt aneinander gereit:

div	{
	transition: width 1s, background-color 2s;
}
