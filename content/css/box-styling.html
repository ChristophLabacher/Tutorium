Box-Styling

Es gibt verschiedene Eigenschaften, mit denen das Aussehen von Block-Elemente beeinflusst werden kann.

Größe

div	{
	width: 100%;
	height: 500px;
	padding: 30px;
	margin: 50px;
}

Laut dem Box-Modell können die folgenden Angaben die Größe eines Elements beeinflussen: width (Breite), heigth (Höhe), padding (Innenabstand) und margin (Außenabstand).

Hintergrundfarbe

div	{
	background-color: rgba(0,0,0,0.5);
}

Mit der Eigenschaft background-color lassen sich (auch halbtransparente) Hintergrundfarben definieren.

Hintergrundbild

div	{
	background-image: url("../imgs/bg.jpg");
	background-repeat: no-repeat;
	background-size: cover;
	background-position: left top;
}

Mit Hilfe der Eigenschaft background-image wird angegeben, welches Bild als Hintergrundbild verwendet werden soll. Dabei wird als Wert url("relativer Pfad") verwendet. Der Pfad zum Bild ist relativ, ausgehen von der Position des Stylesheets anzugeben.

background-repeat gibt an, ob ein Hintergrundbild gekachelt wird. Mögliche Eigenschaften sind no-repeat, repeat-x (horizontale Kachelung), repeat-y (vertikale Kachelung) und repeat (horizontale und vertikale Kachelung).

Mit background-size wird die Größe des Hintergrundbildes angegeben. Dabei können entweder zwei Größenangaben für Höhe und Breite (zum Beispiel 300px 60px oder 600px auto), oder die Werte cover (das Hintergrundbild wird so skaliert, dass die Box immer abgedeckt ist, auch wenn ein Teil vom Bild abgeschnitten wird) oder contain (das Hintergrundbild wird so skaliert, dass auf jeden Fall das gesamte Bild sichtbar ist, auch wenn ein Teil der Box nicht abgedeckt wird) verwendet werden.

background-position gibt für diese Fälle an, welche Punkt der Box und des Hintergrundbilds als übereinander gelegt werden sollen. Dabei wird immer ein vertikaler (left, center, right) und ein horizontaler Wert (top, center, bottom) erwartet. Bei center center, liegt also der Mittelpunkt des Bildes auf dem Mittelpunkt der Box, bei left top die linke obere Ecke des Bilder auf der linken oberen Ecke der Box, usw.

Transparenz

div	{
	opacity: .5;
}

Die Eigenschaft opacity gibt die Transparenz einer Box (inklusive Inhalt) an. Dabei ist der Wert 0 völlig transparent und der Wert 1 vollkommen opak.

Schatten

div	{
	box-shadow: 3px 3px 4px 5px #000000;
}

box-shadow gibt einer Box einen Schatten. Dabei sind müssen mehrere Werte angegeben werden: horizontale Verschiebung, vertikale Verschiebung, Blur, Größe und Farbe.

Blur und Größe sind optional, folgende Angaben sind also auch möglich:

header	{
	box-shadow: 3px 3px #000000;
}

footer	{
	box-shadow: 3px 3px 4px #000000;
}

Konturen

p {
	border-style: solid;
	border-width: 1px;
	border-color: #000000;
}

border-style gibt die Art der Kontur an. Mögliche Werte sind (unter anderem) solid, dotted und dashed.

border-width gibt die Breite der Kontur an.

Mit border-color wird die Farbe der Kontur definiert.

Häufig wird für die drei Werte folgende Kurzschreibweise verwendet:

p {
	border: 1px solid #000000;
}

Es ist möglich, für jeder Seite des Elements eigene Kontureigenschaften zuzuweisen. Dafür wird folgende Syntax verwendet:

p {
	border-top: 1px solid #000000;
	border-bottom: 3px solid #FF0000;
	border-left: 5px dashed #0000FF;
	border-right: 1px solid #00FFFF;
}

Sollen nicht alle drei, sondern nur ein Wert für eine Seite angepasst werden, so ist auch dies möglich:

p {
	border-top-style: dashed;
	border-bottom-width: 2px;
}

Runde Ecken

div	{
	border-radius: 10px;
}

Der Wert der Eigenschaft border-radius gibt den Radius der abgerundeten Ecken einer Box an. Dabei kann auch für jede Ecke, ausgehend von der links oberen im Uhrzeigersinn ein eigener Wert angegeben werden:

div	{
	border-radius: 10px 5px 10px 5px;
}
