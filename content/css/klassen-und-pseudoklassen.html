Klassen und Pseudoklassen

Klassen

Mit den bisher bekannten CSS-Selektoren ist es möglich, das Aussehen jedes Elements, das mit einem bestimmten Tag ausgezeichnet ist, zu beeinflussen (z. B. h1 oder p) oder die Auswahl genauer zu fokussieren, in dem man nur die Elemente, die mit einem bestimmten Tag ausgezeichnet sind auswählt, die sich innerhalb eines anderen, spezifizierten Tag befinden (z. B. header h1, article p strong). Mit Hilfe von Klassen ist es zusätzlich möglich, ganz bestimmte, von Hand im HTML-Code gekennzeichnete Elemente auszuwählen (sowohl abhängig von ihrem Tag (bestimmtest Tag mit bestimmter Klasse), als auch unabhängig davon (bestimmte Klassen)).

Eine Klasse wird als Wert des Attributs class in das öffnende Tag eines HTML-Elements geschrieben. Der Name der Klasse ist frei wählbar, Groß- und Kleinschreibung sind unerheblich. Der Name darf nicht mit einer Zahl beginnen und darf kein Leerzeichen enthalten (es ist Konvention stattdessen einen Bindestrich zu verwenden). Eine Klasse darf beliebig oft im Dokument bei beliebigen Elementen vorkommen. Ein HTML-Element kann auch mehrere Klassen besitzen, diese werden mit Leerzeichen getrennt angegeben.

<h1 class="farbig">Überschrift</h1>

<p class="farbig rechtsbuending<">Beispieltext 1</p>
<p>Beispieltext 2</p>

Um eine Klasse als Selektor in CSS zu verwenden, wird vor den Klassenname ein Punkt geschrieben. Wird nur die Klasse als Selektor angegeben, werden alle Elemente, die mit dieser Klasse gekennzeichnet sind ausgewählt:

.farbig	{
	color: #FF0000;
}

Durch diesen CSS-Code werden alle Elemente mit der Klasse farbig rot dargestellt, im oberen Beispiel die Überschrift und der Beispieltext 1.

Sollen nur Elemente, die mit einem bestimmten Tag ausgezeichnet sind und die Klasse besitzen, selektiert werden, so wird die Klasse direkt an das Tag gehängt:

p.farbig	{
	color: #FF0000;
}

Durch diesen CSS-Code wird jedes <p>-Tag mit der Klasse farbig rot dargestellt, im oberen Beispiel nur der Beispieltext 1.

Es ist wichtig, dass zwischen Tag und Klasse kein Punkt steht, da der Klassenselektor auch mit den aus Einheit 1 bekannten einfachen Selektoren kombiniert werden kann:

p .farbig	{
	color: #FF0000;
}

Durch diesen CSS-Code wird jedes Beliebige Tag mit der Klasse farbig rot dargestellt, dass sich innerhalb eines <p>-Tags befindet.

Pseudoklassen

Zusätzlich zu den Klassen, die man selbst vergeben kann, gibt es so genannte „Pseudoklassen“. Diese Klassen werden nicht von Hand vergeben (es sind also keine normalen Klassen), sondern sie werden bestimmten Elementen in bestimmten Situationen vom Browser automatisch vergeben. Der Selektor von Pseudoklassen beginnt immer mit einem Doppelpunkt (im Gegensatz zu einem Punkt bei Klassen); ansonsten verhält er sich wie ein ganz normaler Klassen-Selektor.

:hover

a	{
	color: #000000;
}

a:hover	{
	color: #FF0000;
}

Die Pseudoklasse :hover erhält ein Element in dem Moment, wenn sich der Mauszeiger darüber befindet.

In diesem Codebeispiel werden zuerst alle Links selektiert und die Textfarbe auf schwarz gesetzt. Mit dem zweiten Selektor werden die Links ausgewählt, die die Pseudoklasse :hover besitzen, und deren Textfarbe auf rot gesetzt. Fährt man nun mit der Maus über einen Link, so erhält er die Pseudoklasse :hover. Da der untere Selektor spezifischer als der obere ist, überschreibt dieser den oberen, und der Text wird rot dargestellt.

:first-child und :last-child

Die Pseudoklassen :first-child und :last-child werden allen Elementen vergeben, die je das erste bzw. letzte Kind ihres Elternelements sind.

:nth-child(n)

Mit dem Pseudoselektor :nth-child(n) lässt sich ein bestimmtes Element auswählen, das das n-te Kinde seines Elternelements ist. Außer Zahlen können auch die Begriffe even (gerade) und odd (ungerade) eingesetzt werden.

:first-of-type, :last-of-type und :nth-of-type(n)

Diese Pseudoklassen verhalten sich genau so, sie werden jedoch an das erste, bzw. letzte, bzw. n-te Element einer Art vergeben.

Beispiel

<ul>
	<li>Erstes Element</li>
	<li>Zweites Element</li>
	<li>Drittes Element</li>
	<li>Viertes Element</li>
	<li>Fünftes Element</li>
	<li>Sechstes Element</li>
	<li>Siebtes Element</li>
	<li>Achtes Element</li>
</ul>
li	{
	background-color: #FF0000;
}

li:first-child, li:last-child	{
	background-color: #00FFFF;
}

li:nth-child(even)	{
	background-color: #0000FF;
}

li:nth-child(6)	{
	background-color: #00FF00;
}
