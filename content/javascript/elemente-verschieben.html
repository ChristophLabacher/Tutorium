Elemente verschieben

.append() & .prepend()

var box1 = $("#box1");
$(".box-container").prepend(box1);
var box2 = $("#box2");
$(".box-container").append(box2);

In Variablen können nicht nur einzelne Werte, sondern auch ganze HTML-Elemente zwischengespeichert werden. Dafür wird das Element per Selektor ausgewählt und einer Variable zugewiesen (es befindet sich in diesem Moment im HTML-Dokument noch an seiner ursprünglichen Position, ist aber mit der Variable verknüpft). Das Element kann dann in ein anderes HTML-Element eingefügt werden, entweder an erster Stelle mit der Funktion .prepend(), oder an letzter Stelle mit der Funktion .append(). In diesem Moment wird das Element aus seiner ursprünglichen Position entfernt und an der neuen Position eingefügt.
