Einführung

JavaScript ist eine 1995 entwickelte Script-Sprache, die es unter anderem erlaubt Websiten auch nach dem sie geladen sind noch zu verändern (also zum Beispiel Klassen zu Elementen hinzuzufügen oder auf Klicks zu reagieren).

In diesem Tutorium wird hauptsächlich jQuery behandelt. Damit man jQuery versteht ist es aber nötig, minimale Grundkenntnisse in JavaScript zu besitzen.
