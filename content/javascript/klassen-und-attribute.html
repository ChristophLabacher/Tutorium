Klassen & Attribute

Klassen

$("main #box").addClass("active");
$("main #box").removeClass("active");
$("main #box").toggleClass("active");

Mit den Funktionen .addClass(), .removeClass() und .toggleClass() können HTML-Elementen Klassen hinzugefügt, entfernt, bzw. getoggelt werden (das bedeutet wenn sie nicht vorhanden sind werden sie hinzugefügt, wenn sie vorhanden sind entfernt). Die Klassen werden in dem Moment verändert, in dem die Funktion aufgerufen wird.

if ($("main").hasClass("active"))	{
	// Dieser Code wird ausgeführt wenn main die Klasse "active" besitzt.
}

Mit .hadClass() kann abgefragt werden ob das selektierte Element eine bestimmte Klasse besitzt. Der Rückgabewert ist dabei entweder true oder false.

Attribute

// Attribut auslesen
var id = $("main").attr("id");

// Attribut setzten
var id = $("main").attr("id", "main-project");

Mit der Funktion .attr() kann der Wert eins Attribut eines Elements ausgelesen oder gesetzt werden.
