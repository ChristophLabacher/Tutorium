$(document).ready(function()	{
	// Hier steht unser Code
});

Der Code in jeder JavaScript-Datei mit jQuery muss zwischen den Zeilen 1 und 3 im oberen Code-Beispiel stehen. JavaScript wird vom Browser so bald wie möglich ausgeführt, auch wenn die HTML-Seite noch nicht bereit ist. Das führt häufig zu Fehlern wenn man versucht Elemente auf der HTML-Seite zu selektieren. Das obere Code-Snippet sagt dem Browser dass er mit dem Ausführen des Codes warten muss, bis die HTML-Seite vollständig aufgebaut ist.
