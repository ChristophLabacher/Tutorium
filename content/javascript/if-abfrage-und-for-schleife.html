if-Abfrage & for-Schleife

if-Abfrage

if (test == true)	{
	console.log("Test ist wahr");
}

Bei einer if-Abfrage wird der Inhalt einer Variable überprüft. Trifft die Bedingung zu, so wird der Code in den geschweiften Klammern ausgeführt (Wenn, dann).

Bedingungen können mit „Vergleichsoperatoren“ formuliert werden.

i == 1 // i ist gleich 1
i != 1 // i ist ungleich 1
i < 1 // i ist kleiner 1
i <= 1 // i ist kleiner-gleich 1
i > 1 // i ist größer 1
i >= 1 // i ist größer-gleich 1

Bedingungen können mit „Logischen Operatoren“ aneinander gehängt werden.

i == 1 && j != 2 // i ist gleich 1 und j ist ungleich 2
i == 1 || j != 2 // i ist ungleich 1 oder j ist ungleich 2
if (test == true && j == 2)	{
	console.log("Test ist wahr und j ist 2");
}

if-else-Abfrage

if (test == true)	{
	console.log("Test ist wahr");
} else {
	console.log("Test ist nicht wahr");
}

Bei einer if-else-Abfrage kann zusätzlich noch Code angegeben werden, der ausgeführt werden soll, wenn die erste Bedingung nicht zutrifft.

if (test == true)	{
	console.log("Test ist wahr");
} else if (test == false && j == 1) {
	console.log("Test ist nicht wahr und j ist 1");
} else {
	console.log("Test ist nicht wahr und j ist nicht 1");
}

Beliebig viele if-Abfragen können aneinander gehängt werden. Dafür wird else if () {} verwendet. Wichtig ist, dass sobald eine Bedingung zutrifft und er entsprechende Code ausgeführt wurde die gesamte Abfrage verlassen wird und die Ausführung des Codes darunter fortgesetzt wird.

for-Schleife

for (var i = 0; i < 5; i++)	{
	console.log(i);
}

Eine for-Schleife erhält in ihrem Aufruf drei Informationen: zuerst eine Variablen-Initialisierung, dann eine Bedingung und zum Schluss ein Stück Code, dass immer am Ende der Schleife ausgeführt wird.

Im Beispielcode bedeutet das folgendes: Eine Variable i wird mit dem Wert 0 initialisiert. Die for-Schleife wird so lange wiederholt, bis i nicht mehr kleiner 5 ist. Am Ende jedes Durchlaufs wird i um 1 erhöht (i++ ist eine Kurzschreibweise für i = i + 1). Jeder Durchlauf der Schleife bewirkt, dass der Wert von i in die Konsole ausgegeben wird. Die Ausgabe sähe also folgendermaßen aus: 

0
1
2
3
4

Nach dem 4 ausgeben wird wird i um 1 erhöht, ist also 5. Die Bedingung i < 5 trifft nicht mehr zu, die Schleife wird also nicht mehr ausgeführt.
