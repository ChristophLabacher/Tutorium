Variablen

var i = 0;
var j = 0.1
var k = "String";
var l = true;

In Variablen können Informationen (Zahlen, Texte („Strings“) oder true/false) gespeichert werden. Bei der Initialisierung (der ersten Nennung) von Variablen muss der Typ der Variable nicht angegeben werden. Alle Variablen werden folgendermaßen initialisiert:

var name = wert;
