Einbindung von JavaScript-Dateien

JavaScript-Dateien müssen genau wie CSS-Dateien im <head>-Bereich des HTML-Dokuments eingebunden werden. 

<script type="text/javascript" src="scripts/scripts.js"></script>

Der JavaScript-Code kann auf beliebig viele JavaScript aufgeteilt werden, so lange diese alle eingebunden werden. Die Dateien werden von oben nach unten ausgeführt.
