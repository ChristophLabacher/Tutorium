Pseudoklassen

Zusätzlich zu den Klassen, die man selbst vergeben kann, gibt es so genannte „Pseudoklassen“. Diese Klassen werden nicht von Hand vergeben (es sind also keine normalen Klassen), sondern sie werden bestimmten Elementen in bestimmten Situationen vom Browser automatisch vergeben. Der Selektor von Pseudoklassen beginnt immer mit einem Doppelpunkt (im Gegensatz zu einem Punkt bei Klassen); ansonsten verhält er sich wie ein ganz normaler Klassen-Selektor.

:hover 

a	{
	color: #000000;
}

a:hover	{
	color: #FF0000;
}

Die Pseudoklasse :hover erhält ein Element in dem Moment, wenn sich der Mauszeiger darüber befindet.

In diesem Codebeispiel werden zuerst alle Links selektiert und die Textfarbe auf schwarz gesetzt. Mit dem zweiten Selektor werden die Links ausgewählt, die die Pseudoklasse :hover besitzen, und deren Textfarbe auf rot gesetzt. Fährt man nun mit der Maus über einen Link, so erhält er die Pseudoklasse :hover. Da der untere Selektor spezifischer als der obere ist, überschreibt dieser den oberen, und der Text wird rot dargestellt.

:first-child und :last-child

Die Pseudoklassen :first-child und :last-child werden allen Elementen vergeben, die je das erste bzw. letzte Kind ihres Elternelements sind.

:nth-child(n)

Mit dem Pseudoselektor :nth-child(n) lässt sich ein bestimmtes Element auswählen, das das n-te Kinde seines Elternelements ist. Außer Zahlen können auch die Begriffe even (gerade) und odd (ungerade) eingesetzt werden.

:first-of-type, :last-of-type und :nth-of-type(n)

Diese Pseudoklassen verhalten sich genau so, sie werden jedoch an das erste, bzw. letzte, bzw. n-te Element einer Art vergeben.

Beispiel

<ul>
	<li>Erstes Element</li>
	<li>Zweites Element</li>
	<li>Drittes Element</li>
	<li>Viertes Element</li>
	<li>Fünftes Element</li>
	<li>Sechstes Element</li>
	<li>Siebtes Element</li>
	<li>Achtes Element</li>
</ul>
li	{
	background-color: #FF0000;
}

li:first-child, li:last-child	{
	background-color: #00FFFF;
}

li:nth-child(even)	{
	background-color: #0000FF;
}

li:nth-child(6)	{
	background-color: #00FF00;
}

transition

Mit der Eigenschaft transition lässt sich einstellen, dass bestimmte Eigenschaften, wenn der entsprechende Wert verändert wird (zum Beispiel da er innerhalb des Pseudoselektors :hover überschrieben wird) nicht einfach „springen“, sondern ein sanfter Übergang zwischen den Werten dargestellt wird.

div	{
	width: 5rem;
	transition: all 1s;
}

div:hover	{
	width: 10rem;
}

Die Eigenschaft transition erhält mindestens zwei, maximal vier Werte. Der erst gibt an welche Eigenschaft einen Übergang erhalten soll. Soll jede Eigenschaft, für die ein Übergang möglich ist, einen Übergang erhalten, so wird all angegeben, ansonsten wird der Name der Eigenschaft angegeben. Als zweites folgt die Dauer des Übergangs. Als optionale dritte Eigenschaft kann die Kurve, in der der Übergang erfolgt angegeben werden; mögliche Werte sind ease (Standardwert), linear, ease-in, ease-out, ease-in-out. Die optionale vierte Eigenschaft gibt an, mit welcher Verzögerung der Übergang beginnen soll, hier wird wiederum eine Dauer angegeben.

div	{
	transition: all 1s ease-in-out 2s;
}

Achtung, Nicht alle Eigenschaften können einen Übergang haben. Hier ist eine Liste der „animatable properties“.

Manchmal sollen verschiedene Eigenschaften verschiedene Übergänge erhalten. Die Eigenschaft transition darf deshalb auch mehrmals angegeben werden:

div	{
	transition: width 1s;
	transition: background-color 2s;
}

transform

Mit Hilfe der Eigenschaft transform können Elemente verschoben, gedreht und verzerrt werden. Die verschiedenen Werte werden dabei als eine Art Funktion mit Argumenten angegeben.

translate

transform: translate(x,y);
transform: translateX(x);
transform: translate3d(x,y,z);

Mit dem Wert translate kann das Element in x- und y-Richtung verschoben werden (anstelle der Platzhalter müssen Größenangaben eingegeben werden). Soll nur eine Richtung genutzt werden, so können die Werte translateX oder translateY verwendet werden. Außerdem ist eine dreidimensionale Verschiebung mit Hilfe des Wertes translate3d möglich, dabei wird zusätzlich ein z-Wert erwartet.

scale

transform: scale(x,y);
transform: scaleX(x);

Mit dem Wert scale kann ein Element vergrößert oder verkleinert, bzw. verzerrt werden. Soll das Element nur in eine Richtung verändert werden, so sind auch die Werte scaleX, bzw. scaleY möglich.

rotate

transform: rotate(angle);

Mit dem Wert rotate kann ein Element gedreht werden. Dabei wird eine Winkelangabe, zum Beispiel „30deg“. Auch eine dreidimensionale Rotation ist möglich, diese ist aber sehr komplex.

Mehrere Werte für die Eigenschaft transform können verwendet werden. Sie werden nacheinander angegeben: 

div	{
	transform: rotate(20deg) scaleX(20px);
}

Insbesondere in Verbindung mit der Pseudoklasse :hover und der Eigenschaft transition ist die Eigenschaft transform ein enorm mächtiges Werkzeug zum Beeinflussen von Elementen auf der Seite: Sie können sich drehen, wenn der Nutzer mit der Maus darüber geht, o.ä.

Browserspezifische Präfixe

Leider sind einige der modernen CSS-Eigenschaften, zum Beispiel transition und transform, noch nicht in allen Browsern vollkommen unterstützt. Deshalb müssen bei diesen Eigenschaften Präfixe verwendet werden, die zeigen dass die Eigenschaft experimentell ist. Jeder Browser erwartet eigene Präfixe, weshalb dann häufig eine lange Liste mit Eigenschaften entsteht. Welche CSS-Eigenschaften Präfixe benötigen: www.shouldiprefix.com. Hier können die entsprechenden Präfixe auch direkt kopiert werden. Alternativ kann der CSS-Code auch einfach hier eingefügt werden: www.pleeease.io/play. overflow

Wenn der Inhalt eines Containers größer ist als der Container selbst, wird standardmäßig der Inhalt trotzdem angezeigt, er überlagert dann unter Umständen andere Elemente. Dieses Verhalten kann mit der Eigenschaft overflow beeinflusst werden. Der beschriebene Standardwert ist visible, mögliche weitere Werte sind hidden (der Inhalt wird am Rand des Containers abgeschnitten), scroll (am Rand des Containers werden Scrollbars angezeigt) und auto (nur wenn der Inhalt tatsächlich größer als der Container ist werden Scrollbars angezeigt).

Soll das Verhalten nur in einer Dimension beeinflusst werden, so kann die Eigenschaft overflow-x, bzw. overflow-y verwendet werden.
